无需使用您的实体电源键即可帮您关闭屏幕（锁屏）。

菠萝锁屏是个干净清爽且快速的实用工具应用，用于帮您在无需使用您的实体电源键的情况下即可关闭您的手机屏幕（锁屏）。这可以帮您延长实体电源键的寿命来避免电源键过早损坏。

此应用使用了安卓系统的无障碍功能来实现无需 ROOT 权限即可进行锁屏。

特性

✓ 一键轻触即可关闭屏幕（锁屏）
✓ 可以帮您在启动器创建专用的快捷方式来进行直接锁屏而无需先进入此应用的主界面
✓ 创建无应用图标角标的快捷方式图标*
✓ 支持跟随系统使用亮/暗色主题
✓ 不需要 ROOT 权限
✓ 没有广告

用法

安装完毕后，您需要启用对应的无障碍权限来使得锁屏功能正常工作。您只需按照应用内的描述指引进行操作即可，无需进行其它额外操作。

请注意，若您重启了手机或因其它原因导致应用被停止时，您需要重新启用其无障碍服务才能正常工作。

您也可以在启动器创建快捷方式来便于您进行锁屏操作，所创建的快捷方式可以使您点击后即进行锁屏，此时您即可不进入主界面就完成锁屏操作了。创建快捷方式的方式是可选的，当您不再希望使用快捷方式进行锁屏时也可直接移除对应的快捷方式。
